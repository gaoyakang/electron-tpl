# electron-tpl

electron快速开发模板

## 1.功能集成
- 工程化：代码格式化✅，Git 工作流✅，vitest测试✅，CI/CD，性能监控和日志，文档
- 功能化：react-router✅，zustand✅，antd/icon✅,  axios✅
- 模块化：app生命周期，window生命周期，创建窗口，窗口设置，ipc通信，快捷键，创建菜单，自动更新，剪贴板

## 2.启动项目

### 2.1 安装依赖

```bash
pnpm install
```

### 2.2 开发环境

```bash
pnpm dev
```

### 2.3 构建项目

```bash
pnpm build:win

pnpm build:mac

pnpm build:linux
```
